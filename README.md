A web-based Employee Leave Management System built using Flask, MySQL, HTML, and CSS. This application allows employees to apply for leave and enables administrators to manage, approve, or reject leave requests efficiently.

Employee
Apply for leave
View leave balance (Casual, Sick, Earned)
Track leave request status (Pending / Approved / Rejected)
View leave history
Receive notifications

Admin
View all leave requests
Approve or reject leave applications
Dashboard with request overview
The system allows the Admin to reassign or allocate another employee to cover duties when an employee is on leave.
